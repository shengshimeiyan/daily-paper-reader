## Original Abstract

Long-form text generation by LLMs often contains factual inconsistencies that accumulate over extended outputs. We present FactCheck-Gen, a post-hoc verification and correction framework that processes generated text segment-by-segment, verifying each claim against reliable sources and correcting errors iteratively. Experiments on LongBench-Align and FActScore-Long show FactCheck-Gen improves factual accuracy by 42% with minimal information loss.
